# TextUtils - Your Text Editor #

### Using Textutils, you can perform various operations on your input text such as: ###

- Remove punctuations
- Convert to uppercase
- Remove new line
- Remove extra spaces
- Remove numbers

### Tech Stack ###

- Python
- Django
- HTML5
- CSS
- Bootstrap
